Semantic Elements

Semantic HTML elements are elements that indicate their purpose via their name. For example, a p tag denotes to both an individual and a browser that the content contained within it is a paragraph. Similarly, a ul or ol tag tells us that the content inside will be a list, either unordered or ordered. In contrast, divs or spans (non-semantic elements) can contain a variety of types of content and therefore reading the tag name does not indicate what content a person or browser will encounter nor how it fits with the rest of the content on the page. Using semantic markup is important because it allows people and browsers to distinguish between different types of content. Semantic elements allow us to focus on using HTML to indicate the importance and meaning of our content and using CSS to define the styles and layouts of our webpages.
